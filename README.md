# KontaTech

O objetivo deste projeto é criar um sistema de gestão financeira pessoal e em grupo que ajude os usuários a controlar suas despesas e alcançar metas de economia. O sistema permitirá que usuários se cadastrem, criem ou participem de grupos (como família ou colegas de república) e visualizem despesas compartilhadas. As funcionalidades do projeto incluem o cadastro e a divisão de despesas, a definição de metas de economia mensais, e a criação de uma lista de desejos (wishlist).

Além disso, o sistema também oferece ferramentas para acompanhamento de investimentos, notificação de gastos excedentes e relatórios financeiros mensais detalhados. Recursos mais avançados incluem um monitoramento em tempo real dos preços de produtos da wishlist, com notificações quando o valor desejado é alcançado. O sistema também calcula quanto tempo levaria para comprar um item da wishlist com base nas metas de economia do usuário. O projeto utiliza Apache Kafka para notificações em tempo real, alertando todos os membros de um grupo sobre novas despesas cadastradas.
