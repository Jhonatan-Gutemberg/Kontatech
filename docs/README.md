# 📂 Estrutura de Diretórios do Projeto KontaTech

## Pastas

### 🗂️ `atas`
Contém registros formais de reuniões da equipe do projeto, incluindo decisões, pendências e próximos passos.  


---

### 🗂️ `diagramas`
Armazena diagramas UML e representações visuais do sistema, como casos de uso e diagramas de classes.  


---

### 🗂️ `imagens`
Contém imagens utilizadas para documentação, apresentações ou prototipagem do projeto.  


---

### 🗂️ `wireframes`
Guarda os protótipos de interface do usuário (UI), representando a estrutura visual das telas do sistema.  


---

## Arquivos

### 📄 `README.md`
Arquivo principal de documentação inicial do repositório. Contém informações básicas sobre o projeto, como propósito, instruções de instalação, uso e tecnologias empregadas.  


---

**Resumo:**
- O diretório possui **4 pastas** principais e **2 arquivos markdown**.
- Ele abrange desde a documentação técnica e atas até diagramas e wireframes do sistema **KontaTech**.
