**1.5 Objetivo do projeto:** Desenvolver um sistema de gestão financeira colaborativa que permita a usuários e grupos controlar despesas, definir metas de economia, monitorar investimentos e acompanhar preços de produtos desejados em tempo real utilizando Apache Kafka.

**1.6 Benefícios que justificam o projeto:**

- Melhoria no controle financeiro pessoal e familiar
- Transparência nas despesas compartilhadas entre grupos
- Redução de conflitos financeiros em grupos (famílias, repúblicas)
- Auxílio no alcance de metas de economia
- Monitoramento inteligente de preços para decisões de compra
- Experiência prática com tecnologias modernas (Kafka, APIs, microserviços)

**1.7 Qualidade esperada do produto final (requisitos de qualidade):**

- Interface responsiva e intuitiva para web e mobile
- Processamento de notificações em tempo real com latência máxima de 6 segundos
- Disponibilidade mínima de 90%
- Segurança de dados financeiros com criptografia SHA-256
- Suporte a até 1000 usuários simultâneos

## **2. ESCOPO PRELIMINAR E PREMISSAS**

**2.1 O que será feito (escopo do projeto)**

- Sistema web e mobile para gestão financeira colaborativa
- Cadastro de usuários e criação/participação em grupos financeiros
- Funcionalidades de cadastro, visualização e divisão de despesas
- Sistema de metas de economia com acompanhamento de progresso
- Wishlist de produtos com monitoramento de preços em tempo real
- Acompanhamento básico de investimentos (ações, fundos, criptomoedas)
- Calculadora de tempo para aquisição de itens da wishlist
- Relatórios financeiros mensais por categoria e membro
- Sistema de notificações em tempo real usando Apache Kafka
- Integração com APIs externas para cotações e preços

**2.2 O que não será feito no projeto (contra-escopo)**

- Integração com bancos para importação automática de extratos
- Sistema de pagamentos ou transferências financeiras
- Consultoria financeira ou recomendações de investimento
- Aplicativo mobile nativo (apenas web responsivo)
- Sistema de chat ou comunicação entre usuários
- Funcionalidades de empréstimos ou financiamentos
- Integração com cartões de crédito

**2.3 Resultados / serviços / produtos a serem entregues**
